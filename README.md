# ServImage

ServImage is an image generation and editing benchmark built from real-world commercial imaging services.

## Dataset

The ServImage dataset is available on Hugging Face:

https://huggingface.co/datasets/FengxianUP/ServImage_Dataset

We will release the training data and models soon.

## Paper

The paper is available on arXiv:

https://arxiv.org/abs/2604.24023v1
